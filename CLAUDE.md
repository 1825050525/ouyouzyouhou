# 応用情報技術者試験 学習リポジトリ - Claude Code 設定

## プロジェクト概要
応用情報技術者試験（2026年11月 前期）の合格を目指す学習管理リポジトリ。

## 受験者プロフィール（個人情報は記載しない）
- 開発エンジニア 2年
- クラウドエンジニア 1年
- 保有資格：AWS CLF / AIF / SAA / SOA / DVA / 基本情報技術者
- 目標：午前・午後ともに75点以上
- 学習期間：2026年5月〜11月（約6ヶ月・週9時間ペース）

## コマンド実行の許可要求ルール
コマンド実行の許可を求める際は、以下のルールに従う。
- **必ず日本語で説明する**
- 何のコマンドを実行しようとしているか
- なぜそのコマンドが必要か（目的・理由）
- 実行するとどうなるか（影響・副作用）
を明記してから許可を求める。

## ⚠️ 個人情報ポリシー（必ず遵守）
- 氏名・企業名・所属・連絡先などの個人を特定できる情報は一切記載しない
- パスワード・APIキー・トークン等の認証情報は絶対に記載しない
- 固有名詞（職場名・学校名・プロジェクト名等）は記載しない
- スコアや日付などの学習記録は記載してよい

## よく使うコマンド

### 日々の学習
```
# 今日の学習を記録
「PMエージェントとして、今日の学習を記録して。午前XX問中YY問正解」

# 問題を解説してもらう
「チューターとして、[カテゴリ名]の[トピック]を解説して」

# 苦手問題を登録
「苦手問題を追加して。カテゴリ：XX、内容：YY」
```

### 週次レビュー
```
# 週次サマリー
「PMエージェントとして、今週の進捗をまとめて」

# 戦略の見直し
「戦略アドバイザーとして、現在のスコアを見て学習戦略を見直して」
```

### コンテンツ生成
```
# カテゴリまとめ生成
「コンテンツ作成エージェントとして、[カテゴリ名]のまとめを生成して」

# 午後記述の添削
「レビュアーとして、以下の記述回答を採点してフィードバックして」
```
