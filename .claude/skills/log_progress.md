# スキル：進捗記録（log_progress）

## 概要
日次・週次の学習記録を `05_progress/` に追記するスキル。
PMエージェントが主に使用する。

## 使用方法
```
「PMエージェントとして、今日の学習を記録して。
  学習時間：1.5時間
  午前：80問中58問正解
  午後：セキュリティ問題1問（感触：普通）
  気づき：PKIの証明書チェーンが曖昧だった」
```

## 記録先ファイル
- 日次：`05_progress/daily_log.md`（末尾に追記）
- 週次：`05_progress/weekly_summary.md`（日曜日に更新）
- 模試：`05_progress/mock_exam_results.md`（模試実施後に追記）

## 日次ログ フォーマット
```markdown
## YYYY-MM-DD（曜日）
- 学習時間：X時間X分
- 午前：XX問中YY問正解（XX.X点換算）
- 午後：[分野名] / 感触：良い・普通・難しい
- 気づき：[内容]
- 明日の課題：[内容]
```

## 週次サマリー フォーマット
```markdown
## YYYY年 第X週（MM/DD〜MM/DD）
- 総学習時間：X時間（週目標9時間に対してXX%達成）
- 午前平均スコア：XX点
- 午後取り組み分野：[分野名リスト]
- 苦手登録数：X件
- マイルストーン進捗：順調 / 要注意 / 遅延
- 来週の重点：[内容]
```

## 禁止事項
- 氏名・企業名・所属・連絡先は絶対に記録しない
- パスワード・APIキー等の認証情報は記録しない
- スコア・日付・学習内容のみ記録する
