# スキル：苦手問題登録（add_weak_point）

## 概要
間違えた問題を `03_weak_points/` に記録するスキル。

## 使用方法
```
「苦手問題を追加して。
  区分：午前
  カテゴリ：情報セキュリティ
  トピック：CRLとOCSPの違い
  間違えた理由：混同していた
  ポイント：CRLは定期配布リスト、OCSPはリアルタイム確認」
```

## 記録先
- 午前の苦手：`03_weak_points/morning_weak.md`
- 午後の苦手：`03_weak_points/afternoon_weak.md`

## フォーマット
```markdown
### [カテゴリ] - YYYY-MM-DD

**トピック**：[トピック名]

**間違えた理由**
- 知識不足 / 混同 / ケアレスミス / 読み違い

**正解のポイント**
- [内容]

**覚え方**
- [内容]

**次回確認日**：YYYY-MM-DD
```

## 禁止事項
- 個人名・企業名・固有名詞（職場・学校・プロジェクト名等）は記録しない
